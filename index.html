About vsrp ot

This is a demo site that shows the text **vsrp ot** in a clean, modern layout. It’s built with HTML5, CSS Grid/Flexbox, and a tiny vanilla‑JS type‑writer animation. Feel free to replace the copy, colours, or add more sections – it’s a solid foundation.

Features

  • Responsive design – looks great on phones, tablets and desktops.
  • Customisable colour palette using CSS variables.
  • Animated background (cheap particle effect).
  • Type‑writer effect for the main heading.
  • Easily extensible – add more pages or sections.

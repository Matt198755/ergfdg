About Void State Roleplay

Void State Roleplay (VSRP) is a FiveM server built around the vMenu admin suite. We focus on realistic role‑play, deep character development, and a friendly community.

Features include:

  • Custom jobs and scripts
  • Robust law‑enforcement system
  • Dynamic economy & vehicle ownership
  • Regular community events
  • Active staff team handling support & moderation

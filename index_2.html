About Void State Roleplay

Void State Roleplay (VSRP) is a premium FiveM server that runs on the powerful vMenu framework. We focus on immersive role‑play, realistic mechanics, and a tight community driven by a passionate staff team.

Key Features

  • Custom jobs and economy system
  • Robust law‑enforcement & gang mechanics
  • Active development and frequent updates
  • Friendly, supportive community
